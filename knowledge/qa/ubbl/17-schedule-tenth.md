# Tenth Schedule — Fire Alarm and Extinguishment Requirements

Referenced by By-laws 225(1), 226A, 227, 228, 237, 238, 239.

## Legend

### Extinguishing System Types (Note 2)
- **A** — Automatic sprinkler system
- **B** — Water spray system
- **C** — Foam system
- **D** — Gaseous system
- **F** — Other automatic extinguishing systems
- **G** — Hose reel system
- **H** — Pressurised hydrant system

### Fire Alarm System Types (Note 3)
- **1** — Automatic fire detection system
- **2** — Manual electric fire alarm system
- **3** — Signal indicator alarm system
- **4** — Public address system
- **5** — Fireman intercom system and fire command centre
- **6** — Automatic fire monitoring system
- **7** — Self-contained smoke detector

## Open Structure / Open Corridor Definitions

- **Open Structure**: Total opening area ≥**40%** of total perimeter wall area; opening length ≥**50%** of floor perimeter
- **Open Corridor**: Total opening area ≥**25%** of total perimeter balcony wall area; opening length ≥**50%** of balcony perimeter
- **Open Sided**: All sides open
- "Opening" must be to outside/unenclosed space; openings <600 mm² or <25mm width are excluded
