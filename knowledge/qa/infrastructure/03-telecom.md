# Telecommunications Infrastructure

## Primary Standard

**MCMC MTSFB TC G007:2016** -- Technical Code for In-Building Fibre Cabling for Fibre-To-The-Premise (FTTP)

## External Building Requirements

- Underground duct routes must be as straight as possible
- Maximum bending radius: **20x duct diameter**
- Segregation of communication facilities from other utilities
- **Private Property Line (PPL)** is the demarcation point between Network Facilities Provider (NFP) and Property Developer

> **Citation**: MCMC MTSFB TC G007:2016

## Internal Building Requirements

| Component | Purpose |
|-----------|---------|
| **Telecommunication Room (TR)** | Main equipment room |
| **Riser** | Vertical cabling pathway |
| **Trunking** | Horizontal cabling containment |
| **Fibre Termination Box (FTB)** | Floor-level fibre termination |
| **Fibre Wall Socket (FWS)** | Unit-level connection point (MDU and SDU) |
| **Optical Network Unit (ONU)** | Signal conversion equipment |

## Developer Responsibilities

- **Property Developers** must provide the telecommunication infrastructure and fibre cabling to ensure end users can subscribe to FTTP services
- Building Management maintains in-building infrastructure after handover
- MSC Malaysia status buildings follow **MSC Malaysia Telecommunication Infrastructure Building Guidelines, Revision 2010**

> **Citation**: MCMC MTSFB TC G007:2016
