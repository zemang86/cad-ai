# MS 1184:2014 -- Accessibility (Universal Design)

## Overview

MS 1184:2014 (2nd revision) is the Code of Practice on Universal Design and Accessibility in the Built Environment. Referenced by UBBL By-Law 34A: all public buildings must comply with MS 1183 and MS 1184.

## Ramp Requirements

| Parameter | Requirement |
|-----------|------------|
| Maximum gradient | **1:12** (1 unit rise per 12 units run) |
| Minimum width | **1,500 mm** clear |
| Landing length (top and bottom) | Minimum **1,200 mm** |
| Edge protection | Minimum **100 mm** raised edges on open sides |
| Surface | Slip-resistant, contrasting colour to landings |

> **Citation**: MS 1184:2014; UBBL By-Law 34A

## Handrail Requirements

| Parameter | Requirement |
|-----------|------------|
| Height | **840 mm to 900 mm** from floor |
| Sides | Required on **both sides** of ramps and stairs |
| Extension | Extend **300 mm** beyond top and bottom of ramp/stairs |
| Diameter | **32 mm to 50 mm** circular profile |
| Material | Non-slip, non-heat-conducting |

> **Citation**: MS 1184:2014

## Accessible Toilet Dimensions

| Parameter | Requirement |
|-----------|------------|
| Minimum floor area | **1,500 mm x 1,800 mm** |
| Door width | Minimum **900 mm** clear opening |
| Door type | Outward-opening or sliding |
| Grab bars | On both sides of WC, **300 mm** from centre |
| WC seat height | **430 mm to 480 mm** from floor |
| Basin height | **750 mm to 800 mm** from floor |

> **Citation**: MS 1184:2014

## Lift Requirements (Accessible)

| Parameter | Requirement |
|-----------|------------|
| Minimum car size | **1,100 mm x 1,400 mm** |
| Door width | Minimum **900 mm** clear |
| Control buttons height | **900 mm to 1,200 mm** from floor |
| Braille and tactile indicators | Required on all control buttons |
| Audible floor announcements | Required |

> **Citation**: MS 1184:2014

## TGSI (Tactile Ground Surface Indicators)

- Warning TGSI (raised dots): at top/bottom of stairs, ramps, platform edges
- Directional TGSI (raised bars): guide along paths of travel
- Colour contrast required between TGSI and surrounding surface
- Width: minimum **300 mm**

> **Citation**: MS 1184:2014
